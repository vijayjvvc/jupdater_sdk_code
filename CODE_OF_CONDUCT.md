# Code of Conduct

## Our Commitment

We want JUpdater to be a welcoming and respectful open-source project.

Everyone participating in this project is expected to show respect and professionalism.

## Expected Behavior

Contributors should:

- Be respectful
- Give constructive feedback
- Accept different opinions
- Focus on improving the project


## Unacceptable Behavior

The following are not acceptable:

- Harassment
- Personal attacks
- Discrimination
- Spam
- Publishing private information


## Enforcement

Project maintainers may remove content or restrict participation when this code of conduct is violated.

Thank you for helping keep the JUpdater community positive.
